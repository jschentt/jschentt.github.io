34_解构赋值的用法了解吗？如何实现对象和数组的嵌套和重命名

一、解构赋值

1、定义

对象和数组是 Javascript 中最常用的两种数据结构，由于 JSON 数据格式的普及，二者已经成为 Javascript 语言中特别重要的一部分。在编码过程中，我们经常定义许多对象和数组，然后有组织地从中提取相关的信息片段。ES6 中添加了可以简化这种任务的新特性：解构。解构是一种打破数据结构，将其拆分为更小部分的过程。

要点

  • 解构—也就是破坏，破坏原来的结构
  • 赋值—当然就是再次赋值
