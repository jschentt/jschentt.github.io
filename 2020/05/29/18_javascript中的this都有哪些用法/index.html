18_javascript中的this都有哪些用法

一、this 的用法

1、基本概念

  • 什么是 this：在调用函数时，引用正在调用函数的对象的关键字
  • 何时使用：只要对象自己的方法，要使用自己的属性时，就用 this

简而言之，就是 function 的 this 永远指向调用它的对象。而鉴于 JS 所谓的“万物皆对象”，这个 this 因此可以是任何物件，比如 Global 对象。

2、使用方法

（1）单独的 this，指向的是 window 这个对象

（2）全局函数中的 this

在严格模式下，this 是 undefined.

（3）作为构造函数调用，this 指代 new 出的对象

所谓构造函数，就是通过这个函数生成一个新对象，这时，this 就指向这个对象。

（4）对象中的方法，该方法被哪个对象调用了，那么方法中的 this 就指向该对象

（5）用 call 与 apply 的方式调用函数

作用是改变函数的调用对象，此方法的第一个参数为改变后调用这个函数的对象，this 指代第一个参数

（6）定时器中的 this，指向的是 window

（7）元素绑定事件，事件触发后，执行的函数中的 this，指向的是当前元素

（8）函数调用时如果绑定了 bind，那么函数中的 this 指向了 bind 中绑定的元素

二、面试题

1、console.log(a.fn());

对象调用方法，返回了一个方法。

2、console.log((a.fn())());

a.fn()返回的是一个函数，()()这是自执行表达式。this -> window

3、console.log(a.fn()());

a.fn()相当于在全局定义了一个函数，然后再自己调用执行。this -> window

4、console.log(a.fn()() == (a.fn())());

5、console.log(a.fn().call(this));

这段代码在全局环境中执行，this -> window

6、console.log(a.fn().call(a));

this -> a
